Instructions:
Enter the grid using space-separated values. Use numbers only:
1 = empty cell, 0 = wall
The top-left cell is the Start, and the bottom-right is the End.
You are allowed to teleport exactly once between any two empty cells.

Example Input:
1 0 1 1
1 1 0 1
0 1 1 1
